THE ISSUE:
═══════════

Microphone dropdown was showing loopback devices
Loopback dropdown was empty

WHY:
The filtering logic was wrong - all devices were being categorized as microphones


THE FIX:
════════

MICROPHONES: Real mics = 1-2 input channels, 0 output
  Examples: [76] Microphone (Sound Blaster Audigy 5/Rx)

LOOPBACK: Input devices = 2+ input channels, 0 output
  Examples: 
    [33] "What U Hear" (Sound Blaster Audigy 5/Rx)
    [31] Voicemeeter Out A2 (8 channels input)
    [34] Voicemeeter Out A1 (8 channels input)
    [114] Stereo Mix (Conexant HD Stereo Mix)
    And many more Voicemeeter options!


HOW TO RUN:
═══════════
