✓ TECHNICAL SPECIFICATIONS
─────────────────────────────────────────────────────────────────────────────

ALGORITHM:
  Clean_Voice[n] = TANH(Microphone[n] - α × Speaker[n-d])
  
  Where:
    • Microphone[n] = Raw microphone input
    • Speaker[n-d] = Known speaker output
    • α = Subtraction strength (user adjustable)
    • TANH = Soft clipping to prevent distortion

PERFORMANCE:
  • CPU Usage: 5-15% on modern systems

✓ TROUBLESHOOTING QUICK REFERENCE
─────────────────────────────────────────────────────────────────────────────

ISSUE: Speaker audio still comes through
  → Increase "Subtraction Strength" slider (1.0 → 1.5 → 2.0)
  → Verify VoiceMeeter is routing correctly
  → Check speaker device is selected in GUI

ISSUE: Your voice is quiet or removed
  → Decrease "Subtraction Strength" slider (1.0 → 0.8 → 0.5)
  → Verify microphone input level
  → Move microphone farther from speakers

ISSUE: No output in Discord/Teams
  → Verify app has correct output device selected
  → Verify Voice Isolator shows "Running" status
  → Run: python list_devices.py to check device numbers

ISSUE: Device list doesn't show loopback
  → Install VB-Audio VoiceMeeter
  → Restart your PC
  → Re-run Voice Isolator

ISSUE: High CPU usage
  → Increase chunk_size in config.json (2048 → 4096)
  → Lower sample_rate if possible (48000 → 44100)
  → Close other audio applications

For more help: See SETUP.md troubleshooting section
