TROUBLESHOOTING QUICK:
══════════════════════

If Voice Isolator shows RED error:
  → Try different device from the list
  → Read error message (tells you what's wrong)

If Green but speaker audio still plays:
  → Increase "Subtraction Strength" slider to 1.5-2.0
  → Check YouTube is routed to VoiceMeeter

If your voice is quiet:
  → Decrease "Subtraction Strength" slider to 0.8

If YouTube doesn't play:
  → Check Volume Mixer: YouTube → VoiceMeeter Virtual Input
  → Restart browser
